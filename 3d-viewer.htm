Model Requirements:
  • Format: glTF (.gltf) or Binary glTF (.glb)
  • LED entities must be named: LED_0, LED_1, etc.
  • Indices should start at 0 and be sequential
Usage:
  1. Load your 3D model file
  2. Review the 3D preview and statistics
  3. Copy coordinates in your preferred format, or click "Send to 2D Mapper"
  4. Import into the 2D mapper or FastLED
